Лабораторная работа №1
Пользователи и группы в linux

Цель: ознакомиться с правилами создания пользователей и групп. Изучить права доступа к обьектам в OC Linux

1.  Информация о пользователях доступна в файле /etc/passwd
Просмотр файла:
vim /etc/passwd
Укзаны мя пользователя, номер пользователя, номер группы, домашний каталог, командная оболочка  

Информация о группах доступна в файле /etc/group
vim /etc/group
Указаны имя групп и номер группы

Закрытая для пользователей информация о паролях доступна для аддминистратора в файле /etc/shadow
sudo /etc/shadow
Имя пользователя, пароль (или "!" в случае блокирования учетной записи), временные настройки учетной записи (время до смены пароля, время жизни записи и т.д.)

Закрытая для пользователей информация о парорлях для групп доступна для администратора в файле /etc/gshadow
"!" - означает отсуцтвие парорля для групп

Права доступа к любому объекту устанавилваются администратором припомощи команды chmod
1 Вариант:
sudo chmod 775 /home/tmp3
Установили полный доступ к каталогу /home/tmp3 для владельц каталога и группы владельцев, а для остальных пользователей - право на просмотр (x) и чтение (r)
2 Вариант 
sudo chmod o+w /hoe/tmp3
устанавливается право для всех пользовтелей на запись в каталог /home/tmp3
sudo chmod g-w /home/tmp3
убрали право на запись у группы владельцев каталога /home/tmp3

Владельца и группу владельцев любого оббъекта администратору можно поменять команды chown sudo:student /home/tmp3/F1
Изменили владельца каталога /home/tmp3/F1 на пользовтаеля tmp4, а в группу владельцев, на группуstudent
Изменили группу владельцев файла /home/tmp3/readme на группу tmp4

Для работы в других (сущестувющих) группах можго использовать команду newgrp
newgrp tmp5
для смены группы необходимо знать пароль группы
Полсе смены группы необходимо получить общий доступ к объектам по группам владельцев каждого объекта 
После завершения операций под общей группы, выход из группы в сво. группу по умолчанию осущ. командой exit
